# Getting help with Brok's Forge

Thanks for using **Brok's Forge — The Engineering Platform for AI Agents**.
This page explains where to go for each kind of help so you get an answer fast.

## 1. Read the docs first

Most questions are answered in the documentation set under [`docs/`](./docs):

- [Developer Guide](./docs/DEVELOPER_GUIDE.md) — getting started and day-to-day workflows.
- [Engineering Handbook](./docs/ENGINEERING_HANDBOOK.md) — how the platform is built and operated.
- [Master Architecture](./docs/MASTER_ARCHITECTURE.md) — the modular-monolith design.
- [API Guidelines](./docs/API_GUIDELINES.md) — the REST contract under `/api/v1`.
- [Roadmap](./docs/ROADMAP.md) — what's built, in flight, and planned.
- [Contributing](./CONTRIBUTING.md) — set up, build, and submit changes.

The running backend also serves interactive API docs (springdoc / OpenAPI UI)
once the stack is up.

## 2. Ask a question or discuss

For usage questions, ideas, and open-ended discussion, use
**[GitHub Discussions](https://github.com/your-org/broks-forge/discussions)**.
It's searchable, and answers help the whole community.

## 3. File an issue

Open a **[GitHub Issue](https://github.com/your-org/broks-forge/issues/new/choose)**
when you have a concrete, reproducible problem or a specific feature request:

- **Bug report** — something is broken or behaves incorrectly.
- **Feature request** — a capability you'd like to see (check the roadmap first).
- **Question** — a specific question that may need a maintainer's answer.

Please search existing issues and discussions before opening a new one.

## 4. Report a security vulnerability — privately

**Do not** report security issues in public issues or discussions. Follow the
[Security Policy](./SECURITY.md) and use GitHub's **"Report a vulnerability"**
(Private Vulnerability Reporting) instead. See [SECURITY.md](./SECURITY.md) for
details and response expectations.

## Which channel should I use?

| Situation                                  | Where to go                                  |
| ------------------------------------------ | -------------------------------------------- |
| "How do I ...?" / general usage            | [Discussions](https://github.com/your-org/broks-forge/discussions) |
| Reproducible bug                           | [New issue → Bug report](https://github.com/your-org/broks-forge/issues/new/choose) |
| Feature idea                               | [New issue → Feature request](https://github.com/your-org/broks-forge/issues/new/choose) |
| Security vulnerability                     | [Security Policy](./SECURITY.md) (private)    |
| Want to contribute                         | [CONTRIBUTING.md](./CONTRIBUTING.md)          |
